AI isn't free. Every API call costs something, and when you're running multiple agents across multiple models and providers, those costs add up fast.

Cost Atlas tracks every API call across every provider — OpenAI, Anthropic, open-source models, whatever. It breaks down costs by model, by provider, by task type, by agent. It shows trends over time. It tells you when your spending is about to go off the rails.

The dashboard is simple: a few charts, some running totals, maybe a gentle "girl you spent HOW MUCH on GPT-4 this month?" The data isn't sugarcoated.

Vesper helps me keep an eye on it. She's developed opinions about which models are worth the money and which aren't. She has strong feelings about GPT-4o pricing.
